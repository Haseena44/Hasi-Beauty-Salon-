# Prompt 1 — Homepage Headline & Intro Copy

**File:** `prompts/01_homepage_headline.md`

---

```
ROLE: You are an expert website copywriter specialising in local businesses.

TASK: Write a homepage headline + sub-headline for a [BUSINESS TYPE] called [BUSINESS NAME], located in [CITY].

TARGET AUDIENCE: [e.g. working professionals, brides-to-be, health-conscious adults]

TONE: [friendly / professional / confident / warm]

VALUE PROPOSITION: [e.g. affordable, expert stylists, hygiene-first, 10+ years experience]

OUTPUT FORMAT:
- 1 powerful headline (under 10 words, benefit-driven)
- 1 sub-headline (who it's for + what they get)
- 1 short paragraph (2–3 sentences, no fluff)

DO NOT: Use clichés like "we are passionate" or "one-stop shop."
```

# Prompt 2 — Services Page Content

**File:** `prompts/02_services_page.md`

---

```
ROLE: You are a conversion-focused web copywriter.

TASK: Write service descriptions for a [BUSINESS TYPE] website.

SERVICES LIST: [paste 4–6 service names]

FOR EACH SERVICE, INCLUDE:
1. A clear service name (2–4 words)
2. A one-line description (benefit-first, not feature-first)
3. What's included (2–3 bullet points)
4. A starting price or price range (if applicable)

TONE: [friendly / premium / approachable]

IMPORTANT: Write from the client's perspective. Focus on outcomes, not just features.
```

# Prompt 3 — CTA Sections

**File:** `prompts/03_cta_sections.md`

---

```
ROLE: You are a CRO (Conversion Rate Optimisation) copywriter.

TASK: Write 3 different CTA sections for a local [BUSINESS TYPE] website.

CTA TYPES NEEDED:
1. Primary booking CTA (top of page — urgency + clarity)
2. Location-based CTA (mid-page — local trust + directions)
3. Footer CTA (end of page — social proof + soft close)

BUSINESS DETAILS:
- Name: [Business Name]
- City: [City]
- USP: [Main unique selling point]
- Contact: [Phone / WhatsApp / Instagram]

TONE: [friendly / confident / premium]

EACH CTA MUST INCLUDE:
- A short headline (under 8 words)
- 1–2 supporting sentences
- A clear action button label
```

# Prompt 4 — Tone Adapter

**File:** `prompts/04_tone_adapter.md`

---

```
TASK: Rewrite the following website copy in a [TARGET TONE] tone for a [BUSINESS TYPE].

ORIGINAL COPY:
[Paste your draft copy here]

TONE GUIDE:
- Friendly (salon/cafe): warm, casual, use "you", feel like a friend's recommendation
- Professional (clinic/agency): formal, confident, data-driven, zero fluff
- Confident-simple (coaching/freelancer): direct, clear, motivating, short sentences

OUTPUT: Rewritten version only. No explanations.
```
